# Cloud DevOps Platform

An end-to-end cloud DevOps platform using Terraform, Kubernetes and CI/CD.

## Overview
This project demonstrates a production-style DevOps platform where infrastructure is provisioned with Terraform, applications are deployed on Kubernetes, and delivery is automated through CI/CD pipelines.

## Tech Stack
- Terraform
- AWS
- Kubernetes
- Docker
- GitHub Actions
- CI/CD

## Features
- Infrastructure as Code with Terraform
- Containerized workloads
- Kubernetes-based deployments
- Automated CI/CD workflows
- Cloud-native platform design

## Workflow
1. Provision cloud infrastructure with Terraform
2. Build and package application with Docker
3. Run CI/CD pipeline through GitHub Actions
4. Deploy workloads to Kubernetes

## Goals
- Demonstrate end-to-end DevOps platform engineering
- Combine cloud, CI/CD and orchestration in one project
- Showcase production-style architecture
